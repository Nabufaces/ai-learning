# Week 4 Lab: Local RAG Search

Run:

```bash
pnpm lab:rag
```

This lab searches the roadmap Markdown files with simple lexical scoring. It is not a production retrieval system; it is a baseline that makes retrieval failures visible before embeddings are introduced.

Experiment with:

- asking for a week number
- asking for an English term such as `MCP`
- asking for a topic that is not in the repository
